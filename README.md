# XpressUTC - App de Cafetería Universitaria 

Aplicación móvil desarrollada bajo metodología ágil para gestionar pedidos y visualizar el menú de la cafetería universitaria. Este proyecto implementa Clean Architecture, consumo de APIs RESTful y servicios en la nube.

##  Arquitectura y Patrones de Diseño
El proyecto sigue el patrón **MVVM (Model-View-ViewModel)** y **Clean Architecture** para garantizar la separación de responsabilidades y un código limpio.

##  Prerrequisitos y Configuración del Entorno
- **Android Studio:** Versión Panda o superior.
- **JDK:** Versión 17 o superior.
- **Git:** Para el control de versiones.

### Variables de Entorno y Seguridad
Por seguridad, las credenciales no están quemadas en el código (No Hardcoding). Debes crear un archivo `local.properties` en la raíz del proyecto y agregar las claves necesarias.

##  Instrucciones de Compilación y Ejecución
1. Clona este repositorio en tu máquina local
2. Abre el proyecto en Android Studio.
3. Espera a que Gradle sincronice todas las dependencias.
4. Selecciona un emulador o dispositivo físico y presiona **Run (Shift + F10)**.
